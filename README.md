# Thalean Relational Theory

Thalean Relational Theory is the umbrella research program for the Thalean body of work.

It studies how identity, meaning, action, and accountability arise through relational structure, admissible transformation, and invariants that survive changed frames.

Core maxim:

Identity is not isolated substance. Identity is relational invariance under admissible change.

Operational maxim:

A thing is what its relations make possible, and what its presence makes possible for others.

## Internal stack

- Thalean Relational Theory: umbrella framework
- Thalean Graph Theory: mathematical graph core
- Existential Relativity: philosophical thesis
- Discrete Action Theory: operational dynamics
- Dynamic Relational Modal Logic: candidate formal language
- CoRI coherence testing: research method
